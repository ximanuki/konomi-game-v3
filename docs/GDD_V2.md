# 🌟 ぷにスタ ～ゆめふわアイドルものがたり～

## Game Design Document V2

**バージョン:** 2.0（このみちゃん専用 完全再設計）  
**対象:** 6歳女児（このみちゃん）  
**コンセプト:** **推しを育てて、シールを集めて、キラキラデビュー**

---

## 💫 このみちゃん分析：なぜこれにハマるのか

### ハマってるもの → 抽出した本質

| 好きなもの | 本質的な欲求 | ゲームに取り入れる形 |
|-----------|-------------|---------------------|
| HANA | キラキラ、憧れ、応援する喜び | 推しアイドルを育てて応援 |
| ハントリックス | 「見つけた！」の興奮 | 隠れシール、秘密の場所発見 |
| 牧場物語 | 毎日のルーティン、成長を見守る | デイリーお世話、レベルアップ |
| たまごっち | 愛着、お世話の責任感 | 推しキャラへのお世話システム |
| シナモロール | ゆるふわ、かわいい、癒し | 世界観・キャラデザイン全体 |
| ぷにぷにシール | 集める、触る、貼る | シール図鑑、ぷにぷにタップ |
| LINE VOOM | 次々見る楽しさ | ショートライブ、サプライズ演出 |
| ドラえもん | ひみつ道具のワクワク | まほうアイテム、変身 |

## 🐰 ゆめふわキャラクター

### デザインコンセプト

**シナモロール × アイドル × パステル**

```
特徴：
- ながい耳（シナモロール風、表情を表現）
- まるい体（ぷにぷに触りたくなる）
- 大きな目（きらきら☆）
- パステルカラー（ピンク、みずいろ、ラベンダー）
- キラキラエフェクト常時
```

### スターターキャラ（最初に選べる3人）

| 名前 | タイプ | 色 | 性格 | 得意 |
|------|--------|-----|------|------|
| **ミルキィ** | ふわふわ系 | ピンク | やさしい、ちょっとドジ | うた |
| **ソラン** | クール系 | みずいろ | おとなっぽい、てれや | ダンス |
| **ポムリン** | げんき系 | きいろ | いたずらっこ、なきむし | おしゃべり |

### 追加キャラ（たまごから生まれる）

| レアリティ | 確率 | キャラ例 |
|-----------|------|---------|
| ⭐ ノーマル | 60% | ココア、モチモチ、フワリ |
| ⭐⭐ レア | 30% | ラベンダー、ミント、シュガー |
| ⭐⭐⭐ スーパーレア | 8% | ルナ（月のうさぎ）、オーロラ |
| 🌈 レジェンド | 2% | ユニ（ユニコーン）、プリズム（虹の妖精） |

## ⭐ シールシステム（コレクション要素）

### これがこのゲームの心臓部

```
すべての行動が「シール」につながる。
シールを集める → 図鑑に貼る → コンプリートを目指す

ボンボンドロップシール、ぷにぷにシールの
「集めたい！貼りたい！触りたい！」を完全再現
```

### シールの種類

| カテゴリ | シール例 | 入手方法 |
|----------|---------|---------|
| **キャラシール** | ミルキィ、ソランなど各キャラ | ステージ、おせわ |
| **おようふくシール** | ドレス、リボン、ティアラ | おきがえ |
| **ステージシール** | 各ステージの記念 | ステージクリア |
| **イベントシール** | 季節、誕生日など | 期間限定 |
| **ひみつシール** | ？？？ | 隠し条件 |
| **でんせつシール** | キラキラ超レア | 超レア条件 |

### シールのレアリティ

| レアリティ | 見た目 | 出現率 |
|-----------|--------|--------|
| ⭐ ノーマル | つやなし | 50% |
| ⭐⭐ キラ | キラキラ | 30% |
| ⭐⭐⭐ スーパーキラ | ホログラム風 | 15% |
| 🌈 ウルトラレア | 虹色に光る | 4% |
| ✨ シークレット | ？で隠れてる | 1% |

### ぷにぷにシール体験

```
シールをゲットしたら...

1. シールが画面に登場（ぽよん！）
```
