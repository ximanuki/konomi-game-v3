## 🎯 なぜマイクラ・牧場物語は中毒性があるのか

### 分析：子供が「やめたくない」と泣く理由

```
❌ ゲームが面白いから
⭕ 自分の世界が生きているから
```

マイクラも牧場物語も、プレイヤーがいない間も世界が存在し続ける感覚がある。
「明日ログインしたら何かが変わっている」という期待感。
これが**中毒性の正体**。

### 5つの中毒性エンジン

| エンジン | 効果 | 牧場物語の例 |
|----------|------|-------------|
| **所有感** | 「これは私のもの」 | 自分の牧場、自分の家 |
| **育成愛着** | 「私が育てた」 | 作物、動物、恋人 |
| **発見興奮** | 「こんなのあった！」 | 隠しエリア、レアアイテム |
| **未完成焦燥** | 「まだやることある」 | 図鑑埋め、関係値MAX |
| **時間連動** | 「今日しかできない」 | 季節イベント、誕生日 |

## 🌱 コアメカニクス：たねシステム

### 「たね」がすべての始まり

```
たねを見つける → 植える → 育てる → 何かが生まれる → 世界が広がる
       ↑                                    ↓
       └──────────── 新しいたねをもらう ←────┘
```

### たねの種類と生まれるもの

| たねの色 | 生まれるもの | 入手難度 |
|----------|-------------|---------|
| 🟢 みどり | 花、草、野菜 | ★☆☆☆☆ |
| 🟤 ちゃいろ | 木、森 | ★★☆☆☆ |
| 🩷 ピンク | かわいいもの（うさぎのおうち等） | ★★☆☆☆ |
| 🔵 あお | 水関連（池、噴水） | ★★★☆☆ |
| 🟡 きいろ | たてもの（家、お店） | ★★★☆☆ |
| 🟣 むらさき | まほう（虹、雲の道） | ★★★★☆ |
| ⭐ きんいろ | でんせつ（城、ドラゴン） | ★★★★★ |

### たねの成長

```
                    リアル時間経過
                         ↓
    🌱 → 🌿 → 🌸 → 完成！
    
    たね   芽    つぼみ  
   (0h)  (1h)   (3h)   (6h)
```

**重要：枯れない！**
- 水やりを忘れても成長が止まるだけ
- 次に水をあげれば続きから成長
- **失敗体験を完全に排除**

### たねの組み合わせ（発見システム）

近くに植えると特別なものが生まれる：

| 組み合わせ | 結果 |
|-----------|------|
| 花 + 花（同じ色） | レインボーフラワー |
| 木 + 木 + 木 | 森ができて動物が住み着く |
| 池 + 花 | 蓮の花、かえるが来る |
| 家 + 家 + 家 | 村になって住人が増える |
| 虹 + 雲 | 空の世界への道 |

**これが「発見興奮」を生む**

## 🐰 住人システム（愛着エンジン）

### 住人が来る条件

```
おうちを建てる → 住人がやってくる → なかよくなる → プレゼントをくれる
                                                    ↓
                                            レアなたね！
```

### 住人のタイプ

| タイプ | 住人例 | 特徴 |
|--------|--------|------|
| もりのどうぶつ | うさぎ、りす、ことり | 森エリアに来る |
| みずのいきもの | かえる、あひる、さかな | 湖エリアに来る |
| まちのひと | ねこ、いぬ、くまのパン屋 | 村を作ると来る |
| まほうのいきもの | ようせい、ユニコーン | 特別な条件で来る |
| でんせつ | ドラゴン、フェニックス | 超レア |

### なかよし度システム

```
はじめまして → しりあい → ともだち → しんゆう → かぞく
    0%          25%        50%       75%      100%
```

なかよし度の上げ方：
- 毎日会いに行く（+5%）
- プレゼントをあげる（+10%）
- お願いを聞いてあげる（+15%）
- 一緒にあそぶ（+10%）

**なかよし度100%で：**
- 名前をつけられる
- 一緒に冒険できる
- 限定たねをくれる
- ずっと一緒（引っ越さない）

### 住人の生活

住人には1日のスケジュールがある：

| 時間帯 | 住人の行動 |
|--------|-----------|
| あさ (6-10時) | 起きてくる、散歩 |
| ひる (10-16時) | 仕事、遊び |
| ゆうがた (16-20時) | 買い物、おしゃべり |
| よる (20-6時) | おうちで休憩 |

**リアルタイム連動で「生きている」感覚**

## 🎮 アクティビティ（遊びの要素）

ミニゲームではなく、**世界の中の自然な行動**として遊びを組み込む。

### 1. 🌧️ みずやり（毎日の基本）

**やること：** 植物に水をあげる

**操作：**
- じょうろをドラッグして植物にかける
- または植物をタップ

**面白さ：**
- 水やりで植物が「ぷるん！」と喜ぶアニメ
- たまに植物から「ありがとう」のプレゼント（たね！）
- 全部に水やりすると「かんぺき！」演出

**報酬：**
- 植物の成長
- ランダムでたね入手
- 「まいにちがんばったで賞」（7日連続で特別たね）

---

### 2. 🙈 かくれんぼ（住人との交流）

**やること：** 隠れている住人を見つける

**発生条件：** 住人がいたずらで隠れる（ランダム）

**操作：**
- 画面をタップ/スワイプして探す
- 住人が隠れていそうな場所（木の陰、花の中、家の裏）をタップ

**面白さ：**
- 見つけた時の「みつけた！」演出
- 住人のかわいいリアクション
- 見つけにくい場所ほどレアなプレゼント

**報酬：**
- なかよし度UP
- たね or アイテム

---

### 3. 💌 おつかい（クエストシステム）

**やること：** 住人のお願いを聞いてあげる

**発生条件：** 住人の上に「！」マークが出る

**クエスト例：**

| 住人 | お願い | 報酬 |
|------|--------|------|
| うさぎ | 「にんじんをそだてて！」 | にんじんのたね |
| くま | 「はちみつがほしいな」 | 木のたね |
| ようせい | 「レインボーフラワーをみせて」 | まほうのたね |

**操作：**
- 条件を満たした状態で住人に話しかける
- アイテムを渡す（ドラッグ）

**面白さ：**
- 住人が「ありがとう！」と大喜び
- クエストを通じてストーリーが進む
- 連続クエストでレア報酬

---

### 4. 🎁 たからさがし（発見の喜び）

**やること：** 地面に埋まっている宝を掘り起こす

**発生条件：**
- 地面に「キラキラ」マークが出現（ランダム）
- 新しいエリアを解放した時

**操作：**
- キラキラをタップ → スコップで掘る演出 → 宝が出てくる

**掘れるもの：**
- たね（確率70%）
- レアたね（確率20%）
- でんせつのたね（確率1%）
- びっくりばこ（確率5%） → 何が出るかお楽しみ
- からっぽ（確率4%） → 次回のたからが良くなる

**面白さ：**
- 「何が出るかな？」のワクワク
- レアが出た時の派手な演出
- 「きょうのうんだめし」感覚

---

### 5. 🎪 おまつり（季節イベント）

**やること：** 季節ごとの特別イベント

**イベント例：**

| 季節 | イベント | 内容 |
|------|---------|------|
| 春 | おはなみ | 桜の木の下でパーティ |
| 夏 | はなび | 夜に花火を見る |
| 秋 | しゅうかくさい | 作物を収穫してコンテスト |
| 冬 | ゆきまつり | 雪だるまを作る |
| 特別 | たんじょうび | このみちゃんの誕生日をお祝い |

**面白さ：**
- その時期しか遊べない限定感
- 限定たね、限定住人
- 住人たちの特別な姿

---

### 6. 🏠 にわづくり（創造の自由）

**やること：** 庭を自由にデザインする

**操作：**
- アイテムをドラッグして配置
- 回転、移動自由
- いつでも変更可能

**配置できるもの：**
- たねから育てたもの（花、木、建物）
- 住人からもらったアイテム
- イベントで入手した限定品

**面白さ：**
- 自分だけのオリジナル庭
- スクリーンショットで保存
- 住人が庭を褒めてくれる
